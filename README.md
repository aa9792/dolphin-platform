# 海豚交友系統

海豚交友系統是一個 AI 陪伴與決策孿生平台模擬器，展示用戶陪伴、決策模擬、陪伴者中心、學校端情緒儀表板與投資方收入模型。

## 線上入口

部署到 GitHub Pages 後，網站會從 `index.html` 進入，並自動開啟 `dolphin_platform.html`。

## 主要檔案

- `index.html`：網站入口
- `dolphin_platform.html`：投資方與合作方展示平台
- `dolphin_demo.html`：用戶互動 Demo
- `dolphin_decision_twin.html`：決策孿生系統說明
- `slide-*.jpg`、`*概念圖*`：展示與簡報素材

## Demo 定位

目前版本是靜態展示原型，適合用於投資簡報、合作洽談與 MVP 驗證。正式上線產品仍需要後端、資料庫、會員系統、AI API、權限控管、個資保護與危機轉介流程。
